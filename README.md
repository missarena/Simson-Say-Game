# Simson-Say-Game
Simon's Say" (also known as "Simon Says") is a classic memory game where players follow a sequence of instructions given by "Simon." The goal is to repeat the actions in the exact order, but only if they start with "Simon says..."
In my game player have to press a key from the keyboard to start the game.once the game get started,there is flashing of the buttons.Noe the player have to press the buttons in the exact order.Once the player get out final score will be displayed on the screen.
